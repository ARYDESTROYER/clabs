# Array Methods Lab

## Problem Statement

Complete a series of tasks that demonstrate your understanding of JavaScript array methods. The exercise covers fundamental array operations including adding elements, removing elements, finding indices, and modifying array contents at specific positions.

## Required Tasks

### Task 1: Add to Array End
- Add element 1 to the end of the array
- Expected result: [1, 2, 3, 4, 5, 1]
- Hint: Use appropriate array method to add at end

### Task 2: Remove from Array End
- Remove the last element from the array
- Expected result: [1, 2, 3, 4, 5]
- Hint: Use appropriate array method to remove last element

### Task 3: Add to Array Beginning
- Add element 0 to the beginning of the array
- Expected result: [0, 1, 2, 3, 4, 5]
- Hint: Use appropriate array method to add at start

### Task 4: Remove from Array Beginning
- Remove the first element from the array
- Expected result: [1, 2, 3, 4, 5]
- Hint: Use appropriate array method to remove first element

### Task 5: Find Element Index
- Find the index of element 3 in the array
- Store the result in variable 'num'
- Expected result: 2 (because 3 is at index 2)
- Hint: Use appropriate array method to find index

### Task 6: Remove Specific Element
- Remove element 3 from the array
- Expected result: [1, 2, 4, 5]
- Hint: Use appropriate method with the index found

### Task 7: Add at Specific Index
- Add element 6 at index 2
- Expected result: [1, 2, 6, 4, 5]
- Hint: Use appropriate method to insert at specific position

## Expected Output Format

Your program should generate output in this format:

```
Original Array: 1,2,3,4,5
Array After Adding Element 1: 1,2,3,4,5,1
Array After Removing Last Element: 1,2,3,4,5
Array After Adding Element 0: 0,1,2,3,4,5
Array After Removing First Element: 1,2,3,4,5
Index of Element 3: 2
Array After Removing Element 3: 1,2,4,5
Array After Adding Element 6 at Index 2: 1,2,6,4,5
```

## Common Array Methods Reference

- `push()`: Adds element(s) to end of array
- `pop()`: Removes last element from array
- `unshift()`: Adds element(s) to beginning of array
- `shift()`: Removes first element from array
- `indexOf()`: Finds index of an element
- `splice()`: Adds/removes elements at specific position
