# Array Operations

## Problem Statement

You need to implement a JavaScript program that performs basic array operations. The program should demonstrate your understanding of array manipulation in JavaScript by:

1. Creating an array with elements 1, 2, 3, 4, and 5
2. Changing the value of the element at index 2 to 6
3. Getting the length of the array

The program should display each step's result in an HTML element with the ID 'output'.

Do not remove the existing HTML elements or change their IDs or edit the existing JavaScript code. You can add new JavaScript code as needed.
