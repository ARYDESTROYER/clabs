# To-Do List

## Overview

This is a simple yet functional To-Do List web application that allows users to manage their tasks through a clean interface. The application is built using HTML, CSS, and jQuery, demonstrating fundamental concepts of DOM manipulation and event handling.

## Features

- Add new tasks to the list
- Mark tasks as complete/incomplete
- Delete tasks from the list
- Clean and responsive user interface

## Requirements

The main functionality is to be implemented in `script.js` using jQuery:

1. **Adding Tasks**

   - Already implemented
   - Tasks are added when the "Add Task" button is clicked
   - Empty tasks are prevented from being added
   - Input field is cleared after task addition

2. **Task Completion**

   - Each task has a "Complete" button
   - Clicking it toggles the task's completed status
   - Completed tasks are visually distinguished

3. **Deleting Tasks**
   - Each task has a "Delete" button
   - Clicking it removes the task from the list
