Display and Visibility Properties

display: inline

This text is inline with inline elements.

display: inline-block

This text is inline with inline-block elements, which have a width and height.

display: block

This text is followed by a block element, which takes up the full width.

visibility: hidden

This text includes a hidden element, which is invisible but still takes up space.

display: none

This text includes a display: none element, which does not appear and takes up no space.
